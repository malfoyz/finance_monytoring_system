# Finance Monitoring System

Информационная система мониторинга финансово-экономических показателей предприятия.

## Возможности

- загрузка финансовых операций из CSV;
- расчет доходов, расходов, прибыли и рентабельности;
- сценарный анализ;
- прогнозирование прибыли с использованием моделей машинного обучения;
- оценка риска финансовой устойчивости;
- формирование AI-заключения через OpenAI API.

## Запуск

```bash
uv sync
uv run streamlit run app.py
```

## Переменные окружения

OPENAI_API_KEY=your_api_key
OPENAI_MODEL=gpt-4.1-mini
